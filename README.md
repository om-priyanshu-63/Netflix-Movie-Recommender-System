# 🎬 Netflix Movie Recommender System

A content-based movie recommendation system built with **Python** and **Machine Learning** that suggests similar Netflix movies and TV shows based on their descriptions, genres, cast, directors, and other metadata using **TF-IDF Vectorization** and **Cosine Similarity**.

## 📌 Project Overview

This project analyzes the Netflix titles dataset and recommends content similar to a selected movie or TV show. By combining multiple text features, the system generates personalized recommendations based on content similarity instead of user ratings.

## 🚀 Features

- Content-based recommendation engine
- Data cleaning and preprocessing
- Missing value handling
- Genre distribution analysis
- Content feature engineering
- TF-IDF text vectorization
- Cosine similarity-based recommendations
- Search recommendations by title
- Recommend content based on watched history

## 🛠️ Technologies Used

- Python
- Jupyter Notebook / Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Project Workflow

1. Load the Netflix dataset
2. Clean and preprocess the data
3. Handle missing values and duplicates
4. Perform exploratory data analysis (EDA)
5. Combine text features into a single content column
6. Convert text into TF-IDF vectors
7. Calculate cosine similarity between titles
8. Recommend similar movies and TV shows
